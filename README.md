# 🚑 ResQNet Website

ResQNet is a modern emergency response and rescue coordination platform designed to help users quickly connect with rescue services during critical situations.

The platform focuses on improving emergency communication, accessibility, and rapid response by providing a centralized digital solution for rescue-related operations.

---

# 🌐 Live Demo

🚀 Live Website: https://resqnet-website-1.onrender.com/

---

# 🌟 Problem It Solves

During emergencies, people often face problems like:

- ❌ Delayed emergency response
- ❌ Lack of proper rescue coordination
- ❌ Difficulty finding nearby help
- ❌ Poor communication during disasters
- ❌ No centralized rescue information system

ResQNet helps solve these issues by providing:

- ✅ Fast emergency service access
- ✅ Centralized rescue coordination
- ✅ User-friendly emergency platform
- ✅ Real-time communication support
- ✅ Better disaster response management

# ⚙️ Installation & Setup

Follow these steps to run the project locally on your machine.

---

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/resqnet-website.git
```

---

## 2️⃣ Navigate to Project Folder

```bash
cd resqnet-website
```

---

## 3️⃣ Install Dependencies

```bash
npm install
```

This command installs all required packages and dependencies.

---

## 4️⃣ Run the Development Server

```bash
npm run dev
```

OR

```bash
npm start
```

(depending on project configuration)

---

## 5️⃣ Open in Browser

After running the command, open:

```bash
http://localhost:5173
```

OR

```bash
http://localhost:3000
```

Make sure your project is uploaded to GitHub.

### 2️⃣ Create a Render Account
